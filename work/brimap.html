NBCI 2.0 Biologist Ranking Information Web Map

As the data analyst for the National Bobwhite Conservation Initiative (NBCI), I created this web map for easy access/interaction with their Biologist Ranking Information (BRI) geospatial data layer especially since NBCI scientific protocol for monitoring of bobwhites requires experimental units, called focal areas, to be developed within High and Medium BRI ranking areas. I developed the front end of the web map using ESRI ArcGIS Javascript API and Dojo. I also configured the GIS Server running ArcGIS for Server 10.x for hosting the BRI tiled map service as well as the Python geoprocessing service that allows for users to request BRI data for download.
